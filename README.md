To run it in your local device/machine please do the following:

Step 1: The FFmpeg
- watch this tutorial : https://www.youtube.com/watch?v=SG1Fc5QB8RE&t=8s
   Note: The app will crash if this step is skipped!

Step 2: Run the .bat file
- run the "Run_Linguahe.bat"
  Notes: - This will install all dependencies. Make sure to follow it
         - Depending on our device it might take more than a minute to complete the downloads
  
Step 3(optional): If you want to run it in your mobile devices
- Connect your phone to the same Wi-Fi.
- Type the http://... link into Chrome.
